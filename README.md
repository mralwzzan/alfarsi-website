# مؤسسة مارس العقارية — Mars Real Estate

موقع عقاري لمؤسسة **مارس العقارية** لعرض العقارات (بيع/إيجار) واستقبال استفسارات العملاء.
مبني باستخدام **React + Vite + Tailwind CSS** مع دعم كامل للغة العربية (RTL) و**Supabase** لإدارة العقارات والاستفسارات.

## المميزات
- صفحة رئيسية احترافية مع شريط بحث (الغرض / النوع / المدينة)
- صفحة عقارات مع فلاتر (بحث، غرض، نوع، مدينة، السعر الأقصى)
- صفحة تفاصيل العقار مع نموذج استفسار يصل عبر واتساب ويُحفظ في لوحة الإدارة
- لوحة إدارة للمالك: إضافة/تعديل/حذف العقارات + استعراض الاستفسارات مع تنبيه لحظي
- تواصل سريع عبر واتساب
- بيانات تجريبية تظهر تلقائياً قبل ربط قاعدة البيانات

## التشغيل محلياً
```bash
npm install      # تثبيت الحزم
npm run dev      # تشغيل خادم التطوير
npm run build    # بناء نسخة الإنتاج في مجلد dist/
npm run preview  # معاينة نسخة الإنتاج
```

## ربط قاعدة البيانات (Supabase)
1. أنشئ مشروعاً على [Supabase](https://supabase.com) واضبط `SUPABASE_URL` و`SUPABASE_ANON_KEY` في `src/lib/supabase.js`.
2. شغّل محتوى الملف `supabase_setup.sql` في **SQL Editor** لإنشاء الجداول والصلاحيات والبيانات التجريبية.
3. أنشئ حساب المالك بنفس البريد المحدّد في `OWNER_EMAIL` (افتراضياً `mr.alwzzan@gmail.com`) للدخول إلى `/admin`.

> قبل ربط قاعدة البيانات، يعرض الموقع عقارات تجريبية تلقائياً حتى تظهر الواجهة بشكل كامل.

## الصفحات
- `/` الصفحة الرئيسية
- `/properties` قائمة العقارات والفلاتر
- `/property/:id` تفاصيل العقار
- `/login` تسجيل دخول الإدارة
- `/admin` لوحة إدارة العقارات (للمالك فقط)
