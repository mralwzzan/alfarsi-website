# مكتب الفارس للمحاماة — Al-Farsi Law Office

موقع تعريفي لمكتب الفارس للمحاماة يعرض الخدمات القانونية والأسعار ويتيح حجز الاستشارات والتواصل.
مبني باستخدام **React + Vite + Tailwind CSS** مع دعم كامل للغة العربية (RTL).

## المحتويات
- شريط تنقّل متجاوب (مع قائمة للجوال)
- قسم رئيسي تعريفي (Hero)
- الخدمات: أحوال شخصية، استشارات تجارية، استشارات عامة
- جدول الأسعار
- نموذج حجز استشارة (Modal)
- قسم «عن المكتب» وإحصائيات
- نموذج تواصل ومعلومات الاتصال
- تذييل (Footer)

## التشغيل محلياً

```bash
npm install      # تثبيت الحزم
npm run dev      # تشغيل خادم التطوير
npm run build    # بناء نسخة الإنتاج في مجلد dist/
npm run preview  # معاينة نسخة الإنتاج
```

## التقنيات
- React 18
- Vite 5
- Tailwind CSS 3
- lucide-react (الأيقونات)
- خط Tajawal من Google Fonts
